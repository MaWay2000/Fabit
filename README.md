# Fabit

Premium static landing page rework for 1HOST.UK.

![Preview](./preview.png)

## Files

- `index.html` contains the page structure and content.
- `styles.css` contains the visual system and responsive layout.
- `app.js` adds reveal animations and the hero glow interaction.

## Preview

Open `index.html` locally, or enable GitHub Pages for this repository to publish it online.
